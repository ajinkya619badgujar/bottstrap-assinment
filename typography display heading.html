Full Stack Web Developer

A full stack web developer is a person who can develop both client and server software.

A full stack web developer is someone who works with the Back End — or server side — of the application as well as the Front End, or client side. Full Stack Developers have to have some skills in a wide variety of coding niches, from databases to graphic design and UI/UX management in order to do their job well.

In addition to mastering HTML andCSS , he/she also knows how to:

  • Program a browser (like using JavaScript, jQuery, Angular , or Vue)
  • Program a server (like using PHP, ASP,Python , or Node)
  • Program a database (like using SQL, SQLite, or MongoDB)
